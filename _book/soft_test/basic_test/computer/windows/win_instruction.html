软件测试教程

Windows-家族

一.windows家族划分:

1. 版本

  • 服务器版本：WINDOWS SERVER 2003 》 WINDOWS SERVER 2008 》WINDOWS SERVER 2012 》 WINDOWS SERVER 2012 R2
  • 客户端版本：XP 》 WIN7 》 WIN 8 》 win 10
  • 操作系统 版本号

2. 字长

  • 32bit(位)/64 bit（位）
  • 衡量 CPU 性能的一个指标，CPU 中每个字长包含二进制位数称为字长。CPU 有 32 位和 64 位，字长越长，精度越高，处理能力越强。
  • X86 是 32 位的操作系统
  • X64 是 64 位的操作系统
  • X86(32 位的操作系统)可以安装在 32 位的 CPU 上，也可以安装在 64 位的 CPU 上。
  • X64(64 位的操作系统)只能安装在 64 位的 CPU 上。

3. 用户和任务

  • 根据同一时间用户和任务的多少：
    • 单用户单任务：指的是同一台计算机在同一时间只能同一个用户使用，早期 DOS,95,98,Me,2000 都是单用户单任务的操作系统。
    • 单用户多任务：XP
    • 多用户多任务：支持多个用户同时使用，有多个任务，现在 WIN7,WIN8,WIN10,WINDOWS SERVER 2008,WINDOWS SERVER 2012.
